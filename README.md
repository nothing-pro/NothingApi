# NothingJs 三维应用程序二次开发接口

本仓库用于定义 NothingJs 三维应用程序接口。

## 命令

```bash
# 安装依赖
npm install -g tyarn --registry https://registry.npm.taobao.org
tyarn --registry https://registry.npm.taobao.org
# 生成文档
tyarn doc
```

## 目录

- core 核心渲染
- earth 地球
- panoramic 全景
- postEffect 图片后期
